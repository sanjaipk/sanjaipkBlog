Introduction:

JavaScript Object Notation or JSON is a way to represent data. It is heavily used in the place of XML for data exchange.

Basic Structure:

JSON can be of one the following 2 kinds: Object Key : value pair surrounded by {{ '{' }} , {{ '}' }} Array Comma separated values
Object and Array can contain another object or array as child element. Any XML can be represented in JSON.

Uses:

JSON is lightweight and simple, it is used as output for services that is consumed by mobile applications. JSON is primarily used in javascripts upon interaction with services. Javascript has a JSON parser which can quickly parse response construct.
Image: Images can be encoded into text using base64 encoding and result text can be pushed into JSON construct.
Binary data: BSON is Binary JSON helps to encode large amount of data to JSON Strings and decode the same in client end. It is supported by most of the programming languages

Serialization and deserialization:

Converting text to JSON and vice-versa is supported by most of the programming languages. Even though JSON is text-based notation, it is advisable to use framework and libraries to avoid serious errors

Consuming JSON:

With consumption of JSON, basic HTTP service methods became popular. It is important to understand the basics of HTTP requests. Client application developers should be aware of all HTTP methods like GET, POST, PUT and DELETE
